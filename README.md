# My Portfolio

**Access here: https://franciscolopes8.github.io/Portfolio/**

This is my personal portfolio. I built it because I wanted an easy way to show my
projects and actually explain them in detail, instead of just linking a bunch of repos
and hoping people dig through the code.

Each project has a short summary, and clicking it opens a deeper write-up with the
architecture and how it all fits together.

Some things to look for:

- A project carousel: click a side card to bring it up front, click it again to open
  the full write-up.
- My skills live inside a terminal window, and it actually works. Click it and type `help`.
- Light and dark mode, with the toggle in the top right corner.

It's just plain HTML, CSS and JavaScript, no build step or frameworks.
